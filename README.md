# Superstore-Sales-Analyse
# 📊 Superstore Sales Analysis

## 📌 Project Overview
This project analyzes Superstore sales data to identify key business insights and improve decision-making.

---

## 🛠️ Tools Used
- Excel
- SQL
- Power BI

---

## 📂 Dataset
- Superstore sales dataset containing orders, sales, profit, and customer details.

---

## 📊 Dashboard Preview

![Dashboard](dashboard.png)

---

## 🔍 Key Insights
- Sales are highest in the Technology category
- West region generates maximum revenue
- Discounts impact profit negatively
- High sales do not always mean high profit

---

## 📈 Features
- Interactive Power BI dashboard
- Sales trend analysis
- Region-wise performance
- Category-wise insights

---

## 📁 Files Included
- `.csv` → Raw dataset  
- `.sql` → SQL queries  
- `.pbix` → Power BI dashboard  

---

## 🚀 Conclusion
This project helps businesses understand sales patterns and improve profitability using data-driven insights.
